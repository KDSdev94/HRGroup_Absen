✅ 1. PERENCANAAN FITUR (Feature Requirements)
A. Fitur Utama

Login user

Admin login (email & password Firebase Auth)

Karyawan login (optional, kalau mau dipisah)

Atau login via Google

Halaman Dashboard Admin

Rekap absensi harian & bulanan

Total hadir, izin, sakit, terlambat

Manajemen Karyawan

Tambah karyawan (nama, NIK, divisi)

Generate QR Code unik per karyawan

Download QR Code

Absensi via Scan QR Code

Menggunakan kamera HP / laptop

Scan QR → kirim data ke Firestore

Tercatat waktu & lokasi (opsional)

Absensi Manual (Optional)

Karyawan login → klik tombol "Check In / Check Out"

Rekap & Laporan

Filter absensi per karyawan / tanggal

Export ke Excel / PDF

Frontend

React + Vite

Tailwind CSS

React Router

QR Scanner Library (misalnya react-qr-reader)

Backend (serverless)

Firebase

Authentication

Firestore Database

Storage (untuk simpan QR Code jika perlu)

✅ 3. ALUR KERJA SISTEM (Flow)
Admin

Login → masuk dashboard

Tambah data karyawan

Sistem generate QR Code

Admin print / bagikan QR Code

Admin melihat rekap absensi

Karyawan

Datang kantor

Menuju tablet / laptop absensi atau pakai HP

Scan QR Code → check-in

Pulang → scan QR Code → check-out
